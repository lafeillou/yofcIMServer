YOFC Messenger 长飞Web应用IM应用管理平台

依赖网易云信IM功能，提供统一的IM用户管理，提供JSSDK供目标应用集成，简单易用

1.项目介绍

yofc Messenger是受到Facebook Messenger的启发开发的一款通用IM应用管理程序。主要是为了智慧检务团队开发的web应用程序快速便捷引入网易云信IM功能的而开发的统一接入平台； 即自研应用不再去分散对接网易云信，而是只需要引入yofc Messenger提供的JSSDK，集成方式就是引入一个<script>标签并做少量配置,大大简化了集成过程，并且在yofc messenger可以做一些定制开发，更方便的管理通讯的各方用户，基于用户数据扩展更多功能。

功能列表：

  • 新增应用，编辑应用信息
  • 注册用户，一个应用下可以新增多个用户，编辑用户注册信息、删除用户

2.开发者须知

需要Fork的库:

  • yofcIMServer(主项目库):  https://github.com/lafeillou/yofcIMServer.git
  • yofcIM(测试项目库):  https://github.com/lafeillou/yofcIM.git
  • 长飞公司归档库:  http://10.80.65.35:8080/gitblit/summary/Pre-Research!yofcloudWebIM.git

开发环境需要的软件:

  • 安装Python，就只用到一个命令：python -m SimpleHTTPServer 8000; 8000为端口号，可改为其他值
  • Mysql数据库，这里推荐安装WampServer,一款集成apache/php/mysql的一键安装软件，这里我们只用到mysql
  • 安装nodejs,由于项目用到了eggjs/vue3等框架开发，因此nodejs版本推荐8.11.x及以上
  • 编辑器推荐VSCode,推荐安装下列插件:eggjs 0.3.1/Vetur 0.14.5

如何运行项目，进行演示？

  • yofcIM项目，请运行如下命令： 
    1. npm i
    2. npm run build
    3. 进入dist文件夹，运行:
      python -m SimpleHTTPServer 9000
    4. 在浏览器中输入:http://localhost:9000 可以查看测试应用
    5. 在应用中，集成IM功能则需要，注意测试作为后台的yofcIMServer要运行，在dist/index.html中加入如下代码，及在发布者网站中，对首页进行修改
      6. 
                    <script src="http://localhost:7001/public/3rd/yofcMessenger-sdk.js"></script>
                    <script>
                        yofcMessenger.init();
                    </script>
  • yofcIMServer项目，请运行如下命令： 
    1. npm i
    2. npm run m:up
      ,注意此时要运行mysql
    3. npm run dev
    4. 在浏览器中输入:http://localhost:7001 可以查看测试应用
    5. 管理员登录账号/密码为:admin/admin
